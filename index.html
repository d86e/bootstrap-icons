bootstrap-icons

Pure css bootstrap icons, no dependencies!

Example:
  <!doctype html>
  <html lang="en">
  <head>
    <meta charset="utf-8">
    <link rel="stylesheet" href="bootstrap-icons.css" />
  </head>
  <body>
    <i class="icon icon-apple"></i>
    <i class="icon icon-arrow-up"></i>
  </body>
  </html>
